# Passport System — The Gents Chronicles

The Passport is the showpiece. Pull it out at a bar. Show someone who you are.

---

## Concept

A digital passport that records every place the Gents have been and every milestone they've reached. Based directly on the physical Gentlemen's Passport concept in the Gentlemen/Passport/ folder — but alive, animated, and always up to date.

---

## Stamp types

### Mission stamps
Created automatically when a Mission entry is published.
One stamp per city visited.

Visual: Circular SVG. Official government aesthetic. Guilloche border (36 decorative dots). City name arced at top, country at bottom, year centered, country code, city-specific landmark silhouette. Gold ink (#B8963E). Generated by Claude Haiku as SVG code.

Examples from existing missions:
- Makarska Entry — June 2021
- Budva Protocol — July 2022
- Milano Dispatch — September 2022
- Belgrade Winter — December 2022
- Split Standard — July 2023
- Budapest Mission — October 2023
- Ljubljana — May 2025

### Achievement stamps
Unlocked by reaching milestones. More ornate than mission stamps. Trophy/crest aesthetic.

| Achievement | Criteria | Stamp name |
|---|---|---|
| First Mission | 1 mission logged | The First Expedition |
| Seasoned Traveller | 5 missions | Five Horizons |
| Grand Tourist | 10 missions | The Grand Tour |
| Steak Connoisseur | 10 steaks logged | The Table |
| Hat-Trick | 3 PS5 session wins in a row | Hat-Trick |
| Return of the Gents | 3rd visit to same city | Return Protocol |
| The Fellowship | All 3 Gents on a mission | Full Complement |
| Anniversary | 1 year since first entry | Year One |
| Night Owl | 20 nights out | The Night Owl |
| The Circle Grows | 10 people added | The Network |

Achievement criteria are evaluated in an Edge Function after each relevant entry creation.

### Diplomatic stamps
Manual. Awarded for extraordinary or legendary moments. Only a Gent can issue a diplomatic stamp to another Gent or to the group.

Examples from existing lore:
- "Quarantine Rebellion" (reference in stamp folder)
- "Winter Operations"
- "The Brotherhood Seal"

## The Passport UI

### Passport cover
Full-screen. Dark green/navy leather texture background. Gold embossed:
```
THE GENTS CHRONICLES
PASSPORT
[Gent's full alias]
[Gold crest/logo]
```

Tap to open → page flip animation into the stamp collection.

### Stamp grid
Masonry or uniform grid of all stamps. Most recent at top.
Each stamp: circular SVG (Claude Haiku-generated), city name, date.
Tap a mission stamp → opens full visa page (`/passport/visa/:stampId`) with stamp, title, lore, photos, AI debrief. Non-mission stamps open a detail modal.

### Passport pages (per mission)
A full "visa page" for each mission. Like the existing physical passport designs.
Shows:
- Country/city stamps for that mission
- Mission title in display font
- Date range
- Entry summary (2–3 lines from the Lore)
- Photos from the mission (small grid)

This is the primary export surface — the "Passport Page" Studio template is essentially this page exported as an image.

### Achievement gallery
Separate tab/section within Passport. All earned achievements displayed as crests/medals. Locked achievements shown as greyed-out silhouettes (with name hidden or revealed — design decision).

## Stamp generation (Claude Haiku → SVG)

Each stamp is generated as SVG code by Claude Haiku and cached in Supabase Storage. Can be regenerated via "Regen Stamps" button on passport page.

Claude generates complete SVG with:
- Double-circle guilloche border (36 decorative dots at r=175)
- City name arced along top via textPath
- Country name arced along bottom
- Year centered in bold serif, country code above
- City-specific landmark silhouette (e.g., Parliament for Budapest, Sebilj for Sarajevo)
- Gold ink color: `#B8963E`
- viewBox: 400x400, transparent background

Storage path: `stamps/{stamp_id}/stamp-{timestamp}.svg`

## Passport export (The Studio)

The Passport Page template exports:
- Format: 1080×1920 (story) — the passport page as a full Instagram story
- Shows the mission's stamps, title, dates, lore excerpt
- White/ivory background with gold accents (Instagram-optimised — lighter than the app itself)
- Brand mark: `THE GENTS CHRONICLES` at the top
